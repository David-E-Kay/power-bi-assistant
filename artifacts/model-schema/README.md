# Model schema snapshots

Parsed `.bim` schema dumps land here as `model-schema-<model>.md`, generated by
`scripts/bim_to_kb_markdown.py`. These are documentation/cache snapshots — retrieve
them by `Grep`-ing for the target object, then a targeted `Read` of that span, not
whole-file reads. Prefer live TE CLI / TOM as the source of truth when available.
